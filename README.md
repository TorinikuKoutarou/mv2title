# mv2title

## 概要
YoutubeなどのMVのタイトルから、曲名を推測するライブラリです。
推測にはローカルLLMを使用します。(Openai互換のAPIが使えれば、オンラインでもできると思います。(未確認))
また、LLMの出力の正当性をある程度、検証します。


## 準備
1.このライブラリを使用したい場所に置く。
2.LM studioやllama.cppなどでホストする。
3.conect.pyのbase_urlを変更する。

## 使い方
main()にリスト形式でタイトルを入力すると、結果がリスト形式で出力されます。

各種オプション
batch_size　入力リストが長い場合に、いくつで分割するかを選択できます。初期値:10
bypass_check　LLMからの返答が妥当かどうかの確認をバイパスするオプションです。初期値:False

## 検証ロジック
以下のすべての項目を満たしたときに、出力が正しいと判断します。
1.入力リストと出力リストの要素の個数が等しい。
2.入力リストの文字列に出力リストの文字列が含まれている。
